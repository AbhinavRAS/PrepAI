# AI Smart Interview Agent

A comprehensive AI-powered interview simulation and evaluation platform that helps students and professionals practice their interview skills with real-time feedback.

## Features

### 🎯 Interview Types
- **College Placement Interviews** - Entry-level technical and HR questions
- **Experienced Professional Interviews** - Advanced technical and behavioral questions
- **IPS/IAS Interviews** - Civil services exam preparation with tough questions
- **Government Job Interviews** - Public sector interview preparation
- **Technical Interviews** - Coding and system design questions

### 📝 Interview Rounds
- **Technical Round (TR)** - Coding, algorithms, system design
- **Managerial Round (MR)** - Leadership, conflict resolution, team management
- **HR Round** - Personal introduction, career goals, behavioral questions

### 🤖 AI-Powered Features
- **Voice Recognition** - Records and transcribes answers
- **Text-to-Speech** - AI asks questions aloud
- **Real-time Evaluation** - Immediate feedback on answers
- **Code Evaluation** - Automated code review for technical questions
- **Video Analysis** - Body language assessment for IPS/IAS interviews
- **Comprehensive Reports** - Detailed performance analysis with scores

### 📊 Evaluation Metrics
- Communication Skills
- Technical Knowledge
- Confidence Level
- Problem-Solving Ability
- Pronunciation and Clarity
- Body Language (for IPS/IAS)
- Code Quality (for technical rounds)
